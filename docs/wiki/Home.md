# Schema - All In One Schema Rich Snippets

**Version:** 1.7.6 | **Author:** [Brainstorm Force](https://www.brainstormforce.com) | **License:** GPL v2

> Improve SEO, elevate rankings, and boost CTR. Add structured data (schema markup) to posts and pages without coding.

---

## What This Plugin Does

Schema - All In One Schema Rich Snippets adds **structured data markup** to your WordPress posts and pages, enabling Google, Bing, Yahoo, and Facebook to display rich results — star ratings, event dates, recipe details, product pricing, and more — directly in search results.

Supported schema types: **Review**, **Event**, **Person**, **Product**, **Recipe**, **Software Application**, **Video**, **Article**, **Service**
